# bfme-resfix

UHD Support on modern Windows OS for the video game trilogy "The Battle for Middle-Earth"

### Run

Java [JRE or JDK](http://www.oracle.com/technetwork/java/javase/downloads/index.html) needs to be installed on your system. 
Afterwards simply execute:

```
bfme-resfix/bfmeResFix_{versionname}.jar with Java
```
